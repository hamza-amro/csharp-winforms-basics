# C# Windows Forms Basics

A beginner-friendly Windows Forms project that demonstrates the fundamentals of desktop application development using C# and WinForms.

The project contains multiple examples that introduce common controls, event handling, form navigation, and MessageBox dialogs. It is intended as a learning resource for developers starting with Windows desktop applications.

---

## Features

### Form Navigation

* Open forms using `Show()`
* Open modal forms using `ShowDialog()`

### TextBox Operations

* Copy text between TextBoxes
* Update text automatically using the `TextChanged` event
* Enable and disable controls
* Show and hide controls
* Change control background colors

### Form Operations

* Change the form title
* Close the current form

### Label Operations

* Update label text dynamically

### MessageBox Examples

* Display a simple message
* Display a message with a custom title
* Display confirmation dialogs
* Use MessageBox icons
* Handle user responses
